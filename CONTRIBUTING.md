# Contribuer à Rootux

Merci de vouloir contribuer à Rootux !

## Comment contribuer ?

- Ouvrez une Issue pour signaler un bug
- Ouvrez une Pull Request pour ajouter des fonctionnalités
- Améliorez la documentation

## Idées de contributions
- Audio spatial (7.1 surround)
- Détection automatique des applications
- Intégration Discord
- Support LADSPA
- Enregistrement audio
